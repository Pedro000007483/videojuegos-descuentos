ste proyecto calcula el precio final de un videojuego aplicando descuentos según las siguientes reglas:

1. Si el precio del videojuego es mayor o igual a Q500, se aplica un descuento del 10%.
2. Si el cliente es miembro VIP, se aplica un descuento adicional del 15% sobre el precio ya rebajado o sobre el precio original.
3. El sistema muestra el precio final que debe pagar el cliente.

## Archivos incluidos
- `descuentos.py`: Código en Python que implementa la lógica de descuentos.
- `diagrama.png`: Diagrama de flujo que representa la lógica del sistema.

## Cómo ejecutar
1. Clonar el repositorio.
2. Ejecutar el archivo `descuentos.py` en Python 3.
